## Packages

### API

- [ketting](https://github.com/badgateway/ketting) - Ketting is a Hypermedia client for javascript
- [twilly](https://github.com/DCtheTall/twilly) - An abstraction layer on top of the Twilio API to make automating customer service interactions much easier! Designed for use with Node.js and Express 
- [autoserver](https://github.com/ehmicky/autoserver) - Create a full-featured REST/GraphQL API from a configuration file

### Analytics

- [Viper](https://github.com/aaronjwood/viper) - Real-time tracking and analytics using Node.js and Socket.IO

### Algorithms

- [average-rating](https://github.com/ndaidong/average-rating) - Calculate average score and rating based on Wilson Score Equation

- [reattempt](https://github.com/wsmd/reattempt) - Give your functions another chance

### Command Line Tools

- [db](https://github.com/infostreams/db) - Version control for databases: save, restore, and archive snapshots of your database from the command line
- [fx](https://github.com/antonmedv/fx) - Command-line tool and terminal JSON viewer
- [xmysql](https://github.com/o1lab/xmysql) - One command to generate REST APIs for any MySql Database.
- [jay](https://github.com/nikersify/jay) - Supercharged JavaScript REPL
- [kosko](https://github.com/tommy351/kosko) - Write Kubernetes manifests in JavaScript

### Command Line Interface

- [oclif](https://github.com/oclif/oclif) - Node.js Open CLI Framework. Built with purple_heart by Heroku.

### Database

- [RxDB](https://github.com/pubkey/rxdb) - RxDB A realTime database for JavaScript Applications
- [Nano-SQL](https://github.com/ClickSimply/Nano-SQL) - Universal database layer for the client, server & mobile devices. It's like Lego for databases.
- [Mongoku](https://github.com/huggingface/Mongoku) - The Web-scale GUI for MongoDB

### Date Time

- [date-fns](https://github.com/date-fns/date-fns) - Modern JavaScript date utility library

### Development

-  [actions-toolkit](https://github.com/JasonEtco/actions-toolkit) - A toolkit for building GitHub Actions in Node.js

- [docz](https://github.com/doczjs/docz) - It has never been so easy to document your things!
- [outline](https://github.com/outline/outline) - The fastest wiki and knowledge base for growing teams. Beautiful, feature rich, markdown compatible and open source. 

### Docker

- [portainer](https://github.com/sohammondal/portainer) - Making Docker management easy

- [levelui](https://github.com/heapwolf/levelui) - A GUI for LevelDB management based on atom-shell.

- [deepmerge](https://github.com/TehShrike/deepmerge) - A library for deep (recursive) merging of Javascript objects

### Realtime

- [hexnut](https://github.com/francisrstokes/hexnut) - Hexnut is a middleware based, express/koa like framework for web sockets
- [hexnut-handle](https://github.com/francisrstokes/hexnut-handle) - Basic middleware abstraction for handling connections and messages
- [hexnut-bodyparser](https://github.com/francisrstokes/hexnut-bodyparser) - Automatically parse JSON messages
- [hexnut-sequence](https://github.com/francisrstokes/hexnut-sequence) - Sequencing middleware for the HexNut framework 
- [hexnut-with-observable](https://github.com/francisrstokes/hexnut-with-observable) - A Hexnut middleware for integrating with rxjs
- [hexnut-restore-connection](https://github.com/francisrstokes/hexnut-restore-connection) - HexNut middleware to restore a lost connection 
- [hexnut-router](https://github.com/francisrstokes/hexnut-router) - Routing middleware for HexNut

### URIs

- [tall](https://github.com/lmammino/tall) - Promise-based, No-dependency URL unshortner (expander) module for Node.js
